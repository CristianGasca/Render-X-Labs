# Render X Labs

## Descripción del proyecto

Transelected by Render X Labs

Es un software de escritorio enfocado en traducir texto del inglés al español en tiempo real. El sistema permite al usuario seleccionar texto con el cursor y recibir una traducción inmediata mediante una ventana flotante, evitando copiar, pegar o cambiar de aplicación.

Además, el sistema contempla comandos de voz y atajos rápidos para mejorar la experiencia de uso.

## Tecnologías

- Python 3.13
- Tkinter
- SpeechRecognition
- Git
- GitHub
- PyAudio
- Google Translate API
- CustomTkinter
